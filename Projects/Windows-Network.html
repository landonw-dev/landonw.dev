Overview

This project recreates an enterprise Windows Server environment within VMware to develop hands-on experience with Microsoft infrastructure technologies. The lab includes domain services, centralized authentication, networking, shared resources, and client management.

Objectives

🌐 Networking

Design an organized IP addressing scheme and configure DNS and DHCP.

👥 Active Directory

Deploy and manage an enterprise Active Directory environment.

🖥 Windows Server

Configure multiple Windows Server roles and services.

📁 Resource Management

Implement centralized file and print services.
