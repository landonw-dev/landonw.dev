Project Overview

Building a Small Enterprise Environment

The goal of this lab was to gain practical experience designing and managing enterprise infrastructure. I created a virtual network environment using VMware and deployed Windows Server roles commonly found in small and medium business environments.

The environment includes centralized authentication, network services, resource sharing, and Windows client management through Active Directory.
