# Personal Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience in Information Technology, cybersecurity, and web development.

## Overview

This portfolio serves as a central hub for my professional work, technical projects, and continuous learning. It was built from scratch using HTML, CSS, and JavaScript with a focus on clean design, performance, and responsiveness.

The website highlights:

* Professional experience
* Technical skills
* Portfolio projects
* Certifications
* Contact information

## Features

* Modern, responsive design
* Smooth scrolling and page animations
* Interactive particle background
* Mobile-friendly layout
* Organized, modular CSS structure
* Clean and maintainable code
* Fast loading performance

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Git
* GitHub
* Cloudflare Pages (deployment)

## Goals

This portfolio is an ongoing project designed to:
